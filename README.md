# Autonomous Work Mesh (AWM)

**Autonomous Work Mesh (AWM)** is a libre, modular framework for designing, coordinating, and evolving autonomous work systems composed of agents, tasks, protocols, and execution contexts.

AWM is intended for:
- Autonomous agents (human, AI, hybrid)
- Cooperative and ethical automation
- Task orchestration without centralized control
- Reproducible, inspectable workflows
- Non-extractive, non-predatory software ecosystems

This project is released into the **public domain** under the Unlicense.
